HALAMAN SELAMAT ULANG TAHUN
============================

CARA MEMBUKA
------------
Klik dua kali file "index.html" — halaman akan terbuka di browser.

CARA MENGGANTI FOTO (paling mudah)
----------------------------------
Buka folder "foto". Di dalamnya ada 6 gambar contoh:
  foto1.jpg, foto2.jpg, foto3.jpg, foto4.jpg, foto5.jpg, foto6.jpg

Timpa (replace) tiap gambar itu dengan foto asli kalian,
TETAP pakai nama yang sama (foto1.jpg dst).
Tidak perlu menyentuh kode sama sekali — halaman langsung memakai foto baru.

Tips: foto berbentuk potret (tinggi) tampil paling rapi.

CARA MENAMBAH MUSIK
-------------------
Buka folder "music", taruh lagu kalian di sana dengan nama:
    lagu.mp3
Halaman otomatis memutarnya. Tombol bulat di pojok kanan bawah
untuk putar / jeda. (Musik mulai begitu halaman disentuh atau
di-scroll, karena browser memblokir musik otomatis sebelum ada
interaksi dari pengunjung.)

CARA MENGGANTI TULISAN
----------------------
Buka "index.html" dengan Notepad / VS Code, lalu ubah:
  - Nama panggilan  -> cari kata "Sayangku"
  - Pesan pembuka    -> cari "Hari ini dunia ikut merayakan"
  - Caption foto     -> cari "Pertama kali kita bertemu" dst
  - Cerita/timeline  -> bagian "Cerita kita"
  - Pesan penutup    -> cari "Semoga tahun ini penuh"
  - Tanda tangan     -> "Dengan cinta, selalu."

CATATAN
-------
Butuh koneksi internet saat pertama dibuka agar font-nya tampil indah
(font diambil dari Google Fonts). Selebihnya berjalan offline.
